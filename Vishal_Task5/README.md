# Sales Prediction Using Python

## Overview
This project uses Machine Learning techniques to predict product sales based on advertising data.  
The model is trained using Python libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn.

The project demonstrates:
- Data loading and preprocessing
- Data visualization
- Linear Regression model training
- Sales prediction
- Performance evaluation using ML metrics

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Algorithm
- Linear Regression

---

## Features
- Advertising dataset analysis
- Data preprocessing and cleaning
- Correlation visualization
- Model training and testing
- Sales prediction system
- Performance evaluation using R2 Score and MAE

---

## Project Workflow
1. Load dataset using Pandas  
2. Analyze dataset information  
3. Visualize relationships between features  
4. Split training and testing data  
5. Train Linear Regression model  
6. Predict sales output  
7. Evaluate model performance  

---

## Dataset Information
The dataset contains advertising expenses spent on:
- TV
- Radio
- Newspaper

Using these values, the model predicts product sales.

---

## Result
The model successfully predicts sales values with good accuracy using Linear Regression techniques.
